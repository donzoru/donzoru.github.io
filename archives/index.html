Hexo Usage

How to Write Article

  1. Edit *.md file in source/_posts with your favourite editor (e.g. Visual Studio Code)
  2. Local debugging
    # clean last build
hexo clean
# generate static pages
hexo g
# run local server 
hexo s
# visit localhost:4000 in browser
  3. publish to github blog
    hexo clean && hexo g && hexo d
